What Is VS Code - Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Console Screen - The console is an operating system window where users interact with the operating system or with a text-based console application by entering text input through the computer keyboard, and by reading text output from the computer terminal. How To Open Console Screen - 1. Right Click And Click On Inspect. 2.Click On Console At Top-Right. How To Print Anything In Console Screen - Write console.log() And Print Anything You Want In Round Brackets
